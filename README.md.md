### 1. Business Understanding

* Compared data structures (structured and unstructured) to understand the source data.

### 2. Data Cleaning & Transformation (SQL)
* Cleaned and transformed data using T-SQL.
* Understood and applied the difference between FACT and Dimension tables.
* Used various SQL concepts for cleaning:
    * Renaming and combining columns
    * Formatting SQL statements and adding comments

### 3. Power BI Dashboard Development
* Loaded the clean data into Power BI.
* Organized and prepared tables for the data model.
* Connected tables to create a relational model.
* Wrote DAX calculation measures (e.g., for `Import Fact_Budget`).
* Designed the final dashboard.
